Markdown 基本语法讲解

Markdown 是一种轻量级标记语言，用于格式化纯文本文档，使其具有良好的可读性和简单的格式。

一、标题


# 一级标题
## 二级标题
### 三级标题


二、文本格式

  *斜体文本* 或 _斜体文本_
  **粗体文本** 或 __粗体文本__
  ***粗斜体*** 或 ___粗斜体___
  ~~删除线文本~~
  
三、列表
---无序列表---------------------------------

* 项目1
* 项目2
  * 子项目2.1
  * 子项目2.2

- 也可以使用减号
+ 或加号

---有序列表---------------------------------

1. 第一项
2. 第二项
   1. 子项1
   2. 子项2
3. 第三项
----------------------------------------

四、链接和图片

[链接文字](https://www.example.com)

![图片描述](图片地址或路径)
![示例图片](https://example.com/image.jpg)

五、引用

> 这是一段引用文本
> 
> 多行引用
> 
>> 嵌套引用

六、代码

行内代码
使用 `code` 标记行内代码

代码块

<pre> ```python def hello(): print("Hello, Markdown!") ``` ```javascript console.log("Hello World"); ``` </pre>

七、表格

| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |

对齐方式：

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 单元格 | 单元格   | 单元格 |

八、分割线

***
---
___

（三个或更多星号、减号或下划线）

九、任务列表

- [ ] 未完成任务
- [x] 已完成任务
- [ ] 另一个任务

十、转义字符

\* 这样就不会被解释为斜体
\# 这样就不会被解释为标题

十一、高级功能

脚注------------------------

这是一个带脚注的句子[^1]。

[^1]: 这是脚注内容。

定义列表--------------------

术语1
: 定义1

术语2
: 定义2
换行-------------------------

<br><br>

Markdown 的优点是**简单易学**、**纯文本兼容**、**跨平台支持**，被广泛应用于文档编写、博客写作、README文件等场景。
